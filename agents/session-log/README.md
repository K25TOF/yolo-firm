# Session Log

This directory contains session logs from Org Learning cycles.

## Format

Each session produces a file: `YYYY-MM-DD-[session-id].md`

See `../protocol.md` for the full session log and minutes format.

## Rules

- Session logs are operational — not versioned in git
- All `.md` files in this directory (except this README) are gitignored
- Logs are append-only — multiple exchanges in one session go in one file
- Session ID format: short descriptive slug (e.g., `vol-filter-audit`, `ema-hypothesis`)
